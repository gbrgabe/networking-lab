# Networking Lab

Networking practice through Packet Tracer simulations and real office lab implementations.

This repository documents my networking journey, from basic configurations to more complex setups. The goal is to build real understanding, not just follow tutorials.

---

## Structure

- [Packet Tracer](packet-tracer/) - Network simulations and troubleshooting
- [Office Lab Networking](office-lab-networking/) - DHCP, DNS Forwarder and real environment configurations

---

## Tools & Technologies

**Simulation:** `Cisco Packet Tracer`

**Real Environment:** `Windows Server 2022` `VirtualBox` `DNS` `DHCP`

---

The office lab networking builds on top of the existing sysadmin-lab environment:

- PC01 as client on 10.0.0.5
- Internal Network (adlab)
